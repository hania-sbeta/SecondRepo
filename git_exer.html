1. use git clone to take a project on Github and put it on your local machine

2. use git status to see the current state of the project, or to see what is not staged

3. rm *.json to remove all of the '.json' files from the current directory

4. rm -rf config to delete a directory named 'config' from your current one.

6. use git remote to specify the parent repo from which you forked your current local copy

7. use git mv config setup

8. use cd .. to move up one directory
